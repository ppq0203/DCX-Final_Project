게임설명

혼자하기

1. 혼자하기를 선택합니다. 2. 난이도, 방크기 선택하면 바로 방으로 입장합니다. 3. 제시된 키워드 클릭하면 문제와 답 입력창이 보입니다. 4. 문제에 포기하기 버튼을 누를 경우 다시 풀 수 없습니다. 5. 나와있는 문제를 모두 풀었을 경우 결과지 확인할 수 있습니다.

같이하기

1. 같이하기를 선택합니다. 2. 개인과 팀, 명수, 난이도를 선택하고 누르면 방이 생성됩니다. 2-2. 만들어진 방의 정보를 확인하고 START 버튼 클릭 3. 유저가 모두 입장하면 방장은 시작버튼을 눌러 게임이 시작됩니다. 4. 주사위는 좌표의 위치를 나타냅니다. 게임 시작후, 문제가 끝날 때 랜덤으로 나타납니다 5. 문제를 먼저 푼 팀/개인이 구역(번호)를 차지 합니다. 6. 가장 많은 구역(번호)를 차지하는 팀/개인이 승리합니다. 7. 게임이 끝나면 최종 결과지를 확인할 수 있습니다.
